This plugin will display a random guitar rhythm in a widget

== Description ==

Random Rhythm will display a random guitar rhythm, now with ajax refresh.

At this time, it can display up to 14641 differents rhythms. 

Guitarists may find new rhythms and/or new exercises.

The plugin will give you only the rhythm, it is up to you to choose the chords.

A template tag `random_rhythm()` may be used also for displaying anywhere.

See a demonstration in my sidebar on <a href="http://www.woodymood-dev.net/cms/wordpress/">my site</a>.

French translation provided and .pot file.

== Installation ==

1. Upload the directory `random-rhythm` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Enter the widget title
1. Activate the widget
